**Tu carrera no tiene fronteras.**

Plataforma que ayuda a jóvenes a descubrir su vocación a través de 8 mundos profesionales, un test vocacional interactivo, y un explorador de universidades con becas reales en Américas, Europa y Asia.

---

## 🚀 Cómo publicar en Vercel (la forma más fácil)

### Opción A: Subir directamente desde la web (sin instalar nada)

1. Ve a **https://vercel.com/signup** y crea una cuenta gratuita (puedes usar tu correo de Google)
2. Una vez dentro, haz clic en **"Add New..."** → **"Project"**
3. Busca el botón **"Import Third-Party Git Repository"** o arrastra esta carpeta completa al área de upload
4. Vercel detectará automáticamente que es un proyecto Next.js
5. Haz clic en **"Deploy"**
6. En 2-3 minutos tendrás tu link en vivo: algo como `brujula-xxx.vercel.app`

### Opción B: Con GitHub (recomendado para cambios futuros)

1. Crea una cuenta en **https://github.com** si no tienes
2. Crea un repositorio nuevo llamado `brujula`
3. Sube esta carpeta al repositorio (puedes arrastrar los archivos desde la web de GitHub)
4. Ve a **https://vercel.com** y conecta tu cuenta de GitHub
5. Importa el repositorio `brujula` y haz clic en "Deploy"
6. Listo — cada vez que actualices el código en GitHub, Vercel lo publicará automáticamente

## 💻 Cómo probar localmente (opcional)

Si quieres ver la página en tu computadora antes de publicar:

1. Instala Node.js desde **https://nodejs.org** (versión 18 o superior)
2. Abre una terminal en esta carpeta
3. Ejecuta: `npm install`
4. Ejecuta: `npm run dev`
5. Abre **http://localhost:3000** en tu navegador
